## 项目名称
> MVP版本撮合引擎
<https://cloud.tencent.com/developer/article/1541271>

## MVP版本需求
1. 支持连续竞价的撮合方式；
2. 支持限价交易、支持撤单；
3. 支持下单和撤单结果的下发；
4. 采用内存撮合技术，在内存里维护交易委托账本；
5. 需要缓存数据，当程序重启时，可以恢复数据；
6. 支持多个交易标的的撮合；7.支持开启和关闭指定交易标的的撮合功能。
